# Web Development Skills

A Claude Code **plugin** that upskills an agent with web/runtime competence — the working
knowledge strong Node, TypeScript, React, Next.js, and React Native engineers apply.

> **Not test-prep** — see [../POLICY.md](../POLICY.md). Only Node.js maps to a certification
> (OpenJS JSNAD/JSNSD — used as scaffold and benchmark); the rest are pure **competence** skills
> with no first-party cert. Skills are guidance, not ground truth — verify against official docs.

## Install

```
/plugin marketplace add toddkasper/expert-skills
/plugin install web@expert-skills
```

## Skills

| Skill | Anchored on |
|---|---|
| [nodejs](skills/nodejs/SKILL.md) | OpenJS Node.js certifications (JSNAD / JSNSD) |
| [typescript](skills/typescript/SKILL.md) | TypeScript Handbook (no vendor cert) |
| [react](skills/react/SKILL.md) | Official React docs (no vendor cert) |
| [nextjs](skills/nextjs/SKILL.md) | Official Next.js docs (no vendor cert) |
| [react-native](skills/react-native/SKILL.md) | React Native + Expo docs (no vendor cert) |
