# Expert Skills for Web & Runtime Competence

A Claude Code **plugin** that upskills an agent with web/runtime competence — the working
knowledge strong Node, TypeScript, React, Next.js, and React Native engineers apply.

> **Independent content — not affiliated with or endorsed by the OpenJS Foundation, Meta,
> Vercel, Microsoft, or Expo.** Product names (Node.js, TypeScript, React, Next.js, React
> Native/Expo) are used here only to identify the subject matter. **Not test-prep** — only
> Node.js maps to a certification (OpenJS JSNAD/JSNSD, used as scaffold and benchmark); the rest
> are pure **competence** skills with no first-party cert. Skills are guidance, not ground truth
> — verify against official docs. Full policy and disclaimer: [POLICY.md](POLICY.md) · License:
> [LICENSE](LICENSE).

## Install

```
/plugin marketplace add toddkasper/expert-skills
/plugin install web-skills@expert-skills
```
