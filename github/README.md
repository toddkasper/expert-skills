# Expert Skills for GitHub Certification Domains (unofficial)

A Claude Code **plugin** that upskills an agent with GitHub practitioner competence, anchored
on GitHub's certification domains. Starting with **GitHub Actions**.

> **Unofficial — not affiliated with, authorized by, or endorsed by GitHub, Inc.** "GitHub" and
> related marks are trademarks of GitHub, Inc./Microsoft, used here only to identify the subject
> matter. **Not test-prep** — the GitHub Actions certification is used only as the scaffold and
> benchmark. Skills are guidance, not ground truth — verify volatile exam/feature facts against
> the official GitHub documentation. Full policy and disclaimer: [POLICY.md](POLICY.md) ·
> License: [LICENSE](LICENSE).

## Install

```
/plugin marketplace add toddkasper/expert-skills
/plugin install github-skills@expert-skills
```
