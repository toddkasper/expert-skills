# GitHub Skills

A Claude Code **plugin** that upskills an agent with GitHub practitioner competence, anchored
on GitHub's certification domains. Starting with **GitHub Actions**.

> **Not test-prep** — see [../POLICY.md](../POLICY.md). The GitHub Actions certification is used
> as the scaffold and benchmark for this skill, not the product. Skills are guidance, not ground
> truth — verify volatile exam/feature facts against the official GitHub documentation.

## Install

```
/plugin marketplace add <this-repo-git-url>
/plugin install github@expert-skills
```

## Skills

| Credential | Skill |
|---|---|
| GitHub Certified: GitHub Actions | [github-actions](skills/github-actions/SKILL.md) |

*GitHub Foundations, Advanced Security, Administration, and Copilot can be added later.*
