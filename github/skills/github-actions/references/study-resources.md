## Exam Logistics Reference

All logistics are from the Microsoft Learn certification page (updated Feb 2026). Verify before registering — delivery migrated from PSI to Pearson VUE in July 2025.

- **Exam code:** GH-200
- **Time limit:** 100 minutes
- **Passing score:** 700 / 1000
- **Cost:** varies by region; set at registration via Pearson VUE (widely cited as $99 USD; confirm at scheduling)
- **Retake policy:** retake 24 hours after first attempt; subsequent retake intervals vary (see official retake policy)
- **Languages:** English, Spanish, Portuguese (Brazil), Korean, Japanese (non-English candidates may request +30 min)
- **Delivery:** Pearson VUE, proctored online or test center
